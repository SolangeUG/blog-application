A Google App Engine Blog Application

This web project is about building, in a step-by-step fashion, a fully-functional blog that has user registration, submitting links, security, caching, and viewing everything together. The final product will be deployed online via Google App Engine.

Languages - Libraries - Tools

  • Python 2.7.14
  • Jinja2 templating engine
  • Google App Engine
  • Any code/text editor (atom) or IDE (PyCharm) of your choice.

Deploying the application

Google App Engine allows to build scalable web and mobile backends in any language on Googleâ€™s infrastructure.
In this project, we use the original local development App Engine SDK for Python. The latest version - 1.9.66 - can be downloaded here.
